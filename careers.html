Solution Architects

Apex, NC · Various unanticipated U.S. locations · Full-time, Permanent
Now Hiring

Position

Design, Manage and implement cloud native solutions, API management, Security and cloud-to-cloud integration across both public and Private clouds.

Duties

  • Analyze, plan, design, and execute migration from monolithic to microservice cloud migrations using NodeJS, Java/JEE, Spring Boot, web flux, JPA, Hibernate, and Spring security.
  • Build CI/CD and automation pipelines in Aws DevOps (ADO) and Jenkins platforms, encompassing source code management, configuration management, automation, and containerization using Docker and Kubernetes and Terraform.
  • Integrate OpenSearch with Kafka and Flink to create a seamless data pipeline, enabling real-time indexing and search capabilities for high-velocity data streams.
  • Deploy and configure Argo Workflows on Kubernetes clusters to automate and scale workflow execution.

Requirements

Master's degree in Science, Technology, or Engineering (any) is required.

Job Site & Travel

Apex, NC and various unanticipated locations throughout the U.S.
